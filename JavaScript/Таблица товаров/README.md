# Фильтр товаров

Веб-приложение для фильтрации и отображения товаров по заданному диапазону цен с использованием внешнего API.

##  Требования
- Современный браузер (Chrome, Firefox, Edge)
- Доступ к API `http://exercise.develop.maximaster.ru/service/products/`
- Отключение CORS для локального тестирования

##  Быстрый старт
1. Клонируйте репозиторий или скачайте `index.html`
2. Откройте файл в браузере с отключенным CORS:
